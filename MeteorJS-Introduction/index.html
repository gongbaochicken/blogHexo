Introduction to Meteor JS

Posted on 2018-02-01 | In Web | | Page View

I would like to share some interesting topics, and this post was originally shared to the SAP internal tech blog in 2017. This post is a brief introduction of Meteor and how we use it to build a simple application Tiny URL, like google URL Shortener, in just 20 minutes.

Meteor[1] is a real-time, easy-use full-stack JavaScript web/mobile platform with data synchronization of client/server. It is on the top of Node.js and MongoDB, which means one language JavaScript can be used in both front-end client and back-end server. In one word, it saves a lot of my time and it keeps its promise of optimizing for developer happiness.

The overall architecture is shown below [2].

For view part, it originally supports Blaze, their native front-end template rendering engine. Now, it supports the most popular frameworks: React and Angular. Compared with Angular, ReactJS is regarded as a more suitable view framework working with Meteor, because it is flexible (Virtual DOM), fast rendered, and enforces a lot of rules and guidelines that make it more clear what the “right” way to do any given task is.

For the data storage, Meteor uses MongoDB collection in server side and a mini-mongo as cache in client side. The communication between them is based on DDP, the Distributed Data Protocol, which is a protocol for fetching structured data from the server, and receiving live updates when that data changes. Meanwhile, Livequery makes real-time updates more efficiently. Therefore, developers can see how code changes will affect the app behavior in a pretty responsive style, and thanks to mini-mongo on the client side, developers can easily query DB from clients during debug.

Besides, Meteor community offers meteor package management (https://atmospherejs.com) like NPM, and a handful of useful packages can easily be installed with “meteor add” or “npm install” commands. More surprising and powerful tools/features can be found if you try it on your own.

Installation

First, make sure you have installed the latest version of Node and NPM. You can install Meteor by using curl:

curl https://install.meteor.com/ | sh

If you are using Windows, please check Meteor official site: https://www.meteor.com/.

Take Away

Now, the web is becoming an increasingly real-time environment. With the growth of online applications like real-time collaboration tools, multiplayer web games, and more, the need for real-time friendly applications is increasing. Meteor is just for that.

It is very convenient and efficient, and we can build real-time web app prototypes in a short time. Also, Meteor uses because developers don’t rely on multiple backend languages, which reduces the learning curve to some extent. Since it supports both React and Angular, and follows the best convention, it can also guarantee the good code structure of the projects.
